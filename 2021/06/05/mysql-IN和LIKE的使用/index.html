一、运算符in的使用

运算符 IN 允许我们在 WHERE 子句中过滤某个字段的多个值。
==where子句使用in语法==
SELECT column_name FROM table_name WHERE column_name IN(value1, value2, …)

适用于多条件查询，使用时更加方便简洁，比如同时查询ID=1,ID=2,ID=3的员工：
select * from employee where id in(123);
